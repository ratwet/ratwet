## 💻 Featured Projects

### 🖱️ [Gesture Cursor Control](https://github.com/ratwet/gesture-cursor-control)
> *Real-time, touchless mouse control using a standard webcam natively on Linux Wayland.
- **Tech Stack:** Python, MediaPipe, OpenCV, Linux evdev/uinput
- **Impact:** Achieved ~40 FPS with <25 ms latency. Engineered a custom weighted moving-average smoother reducing jitter by ~60%. Detects 21 hand landmarks mapping 5 gestures to system events.

### 🤖 [ESP32 WiFi Soccer Robot](https://github.com/ratwet/esp32-soccer-robot)
> *Competitive 4-wheel differential-drive robot built for the CUJ Robotics Competition.
- **Tech Stack:** C++/Arduino, ESP32, HTML/CSS/JS
- **Impact:** Controlled wirelessly via a custom browser-based gamepad (no app needed). Features multi-touch support, live turn-speed tuning, and direction-balance trim driven by dual L298N H-bridge motors with PWM at 240 MHz.
